# 🤖 Roblox AI Generator

## Setup

1. **Auf Replit:**
   - Fork diesen Repo
   - `.env` Datei erstellen
   - `OPENAI_API_KEY=dein_key` hinzufügen
   - Run drücken!

2. **In Roblox Studio:**
   - Hauptscript in `ServerScriptService` einfügen
   - Im Spiel: `/ai [befehl]`
   - Automatisch ausgeführt! ✅

## Beispiele

```
/ai create a red part in the middle
/ai make a jump function
/ai print hello world
```
